## 快速开始

### 系统要求

- **操作系统**: Windows 10/11, macOS 10.14+, Linux
- **内存**: 至少 4GB RAM (推荐 8GB)
- **磁盘空间**: 至少 500MB 可用空间

### 安装

1. 从 [GitHub Releases](https://github.com/your-repo/study-aid/releases) 下载最新版本
2. 运行安装程序
3. 启动应用程序

### 首次运行

1. 启动应用后，您将看到主界面
2. 左侧为导航栏，包含所有功能模块
3. 右侧为内容区域，显示当前选中的功能

## 功能介绍

桌面学习助手是一款功能强大的学习辅助工具，主要功能包括：

- 📝 **笔记管理** - 创建、编辑、搜索笔记
- 🎤 **语音识别** - 实时语音转文字
- 📸 **截图标注** - 屏幕截图和标注工具
- 📄 **文档学习** - PDF/DOCX/TXT 文档解析和AI辅助学习
- ⚙️ **应用设置** - 主题、快捷键、AI服务配置

## 笔记管理

### 创建笔记

1. 点击左侧导航的"笔记管理"
2. 点击"新建笔记"按钮
3. 输入笔记标题
4. 使用富文本编辑器编写内容
5. 笔记会自动保存

### 编辑笔记

1. 在笔记列表中点击要编辑的笔记
2. 修改标题或内容
3. 使用编辑器工具栏进行格式化：
   - **粗体**: Ctrl+B
   - **斜体**: Ctrl+I
   - **下划线**: Ctrl+U
   - **标题**: Ctrl+Alt+1/2/3

### 搜索笔记

- 在搜索框中输入关键词
- 支持标题和内容搜索
- 搜索结果实时更新

### 笔记标签

- 创建笔记时可添加标签
- 点击标签可筛选相关笔记
- 支持多个标签组合

## 视频学习

### 语音识别

1. 进入"视频学习"页面
2. 点击"语音识别"按钮
3. 点击"开始录制"按钮
4. 对着麦克风说话
5. 识别结果会实时显示
6. 点击"停止录制"结束

**提示**:
- 确保麦克风权限已授予
- 在安静环境下使用效果更佳
- 支持普通话识别

### 截图功能

#### 全屏截图

- 点击"全屏截图"按钮
- 或使用快捷键 `Ctrl+Shift+S`（默认）
- 截图会自动保存到截图面板

#### 区域截图

- 点击"区域截图"按钮
- 选择要截取的屏幕
- 拖动选择区域
- 确认截图

#### 截图标注

1. 在截图预览面板点击"标注"
2. 使用工具栏：
   - 🖌️ 画笔 - 自由绘制
   - ⬜ 矩形 - 绘制矩形
   - ➡️ 箭头 - 添加箭头
   - 🔤 文字 - 添加文字
3. 点击"撤销"可取消上一步操作
4. 点击"清空"清除所有标注
5. 点击"保存"保存标注后的截图

### 图片识别

1. 点击"图片识别"按钮
2. 上传或拖拽图片到上传区域
3. 选择识别类型：
   - OCR 文字识别
   - 图像理解
4. 点击"开始识别"
5. 查看识别结果
6. 可编辑结果或保存到笔记

## 文档学习

### 导入文档

支持的文档格式：
- **PDF** (.pdf)
- **Word 文档** (.docx)
- **文本文档** (.txt)

导入步骤：
1. 进入"文档学习"页面
2. 拖拽文档到上传区域
3. 或点击上传区域选择文件
4. 等待文档解析完成

### AI 辅助学习

#### 生成知识目录

1. 打开一个文档
2. 点击"生成知识目录"按钮
3. AI 会自动分析文档结构
4. 生成层级化的知识目录树

#### 标记要点

1. 打开一个文档
2. 点击"标记要点"按钮
3. AI 会提取文档中的关键要点
4. 按重要性分类显示

#### 标注疑难点

1. 打开一个文档
2. 点击"标注疑难点"按钮
3. AI 会识别文档中的难点
4. 提供详细的解释说明

### 文档操作

- **复制内容**: 点击"复制"按钮复制文档内容
- **保存到笔记**: 将AI生成的内容保存为新笔记
- **切换标签**: 在不同AI功能间切换

## 设置配置

### 外观设置

**主题模式**：
- 亮色模式 - 适合白天使用
- 暗色模式 - 适合夜间使用
- 跟随系统 - 自动跟随系统主题

### 快捷键配置

| 功能 | 默认快捷键 | 说明 |
|------|------------|------|
| 全屏截图 | Ctrl+Shift+S | 快速截取全屏 |
| 语音录制 | Ctrl+Shift+V | 快速开始语音录制 |

**自定义快捷键**：
1. 进入"设置" > "快捷键"
2. 修改快捷键输入框
3. 点击"注册快捷键"

### AI 服务配置

#### 语音识别服务

- **服务商**: 科大讯飞、Azure Speech、腾讯云
- **配置项**: API 密钥、应用 ID

#### 图像识别服务

- **服务商**: 百度OCR、通义千问VL、腾讯OCR
- **配置项**: API 密钥、Secret Key

#### 内容生成服务

- **服务商**: DeepSeek、Kimi、OpenAI GPT-4
- **配置项**: API 密钥、API 端点

**配置步骤**：
1. 进入"设置" > "AI 服务"
2. 选择服务商
3. 输入相应的 API 密钥
4. 点击"测试所有服务"验证配置
5. 点击"保存配置"

### 存储设置

- **数据存储路径**: 应用数据保存位置
- **截图保存路径**: 截图文件保存位置

**更改存储路径**：
1. 进入"设置" > "存储"
2. 点击"选择"按钮
3. 选择新的文件夹
4. 点击"保存配置"

## 常见问题

### Q: 如何备份我的笔记？

A: 笔记数据保存在应用的数据目录中。您可以：
1. 进入"设置" > "存储"
2. 点击"打开数据文件夹"
3. 复制整个数据文件夹进行备份

### Q: 语音识别不准确怎么办？

A: 尝试以下方法：
- 确保环境安静
- 靠近麦克风说话
- 语速适中，发音清晰
- 检查麦克风设置

### Q: 截图快捷键不工作？

A:
1. 检查快捷键是否被其他应用占用
2. 进入"设置" > "快捷键"
3. 点击"注销快捷键"后重新注册
4. 尝试使用其他快捷键组合

### Q: AI 功能无法使用？

A:
1. 检查 API 密钥是否正确配置
2. 确认网络连接正常
3. 进入"设置" > "AI 服务"
4. 点击"测试所有服务"检查连接状态

### Q: 应用占用空间过大？

A:
1. 进入"设置" > "存储"
2. 查看存储信息
3. 可以更改数据存储路径到其他磁盘
4. 定期清理截图文件

### Q: 文档解析失败？

A:
1. 确认文档格式受支持（PDF/DOCX/TXT）
2. 检查文档是否已损坏
3. 尝试使用其他工具转换文档格式
4. TXT 文档确保编码为 UTF-8 或 GBK

## 故障排除

### 应用无法启动

**症状**: 双击应用图标无反应

**解决方法**:
1. 检查系统任务管理器，确认没有残留进程
2. 重启计算机
3. 重新安装应用
4. 查看日志文件了解详细错误

### 界面显示异常

**症状**: 界面错位、显示不全

**解决方法**:
1. 尝试切换主题（亮色/暗色）
2. 调整窗口大小
3. 重置设置（删除应用数据文件夹中的 settings.json）
4. 更新到最新版本

### 保存失败

**症状**: 笔记无法保存

**解决方法**:
1. 检查磁盘空间是否充足
2. 确认数据目录有写入权限
3. 检查文件名是否包含特殊字符
4. 查看控制台错误信息

### 性能问题

**症状**: 应用运行缓慢

**解决方法**:
1. 关闭其他占用资源的程序
2. 清理旧笔记和截图
3. 更新到最新版本
4. 增加系统内存

### 网络问题

**症状**: AI 功能无响应

**解决方法**:
1. 检查网络连接
2. 确认 API 服务状态
3. 验证 API 密钥有效性
4. 检查防火墙设置
