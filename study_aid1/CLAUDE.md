## Project Overview

This is a **桌面学习助手** (Desktop Learning Assistant) - a desktop application designed to help users learn from videos and documents. The project is currently in the planning/initialization phase.

## Core Features

### 1. Video Learning (视频学习)
- **Speech Recognition**: Real-time voice-to-text conversion for generating classroom notes
- **Screenshot Capture**: One-click screenshot with automatic saving to classroom notes
- **Image Recognition**: One-click image recognition with content extraction into classroom notes

### 2. Document Learning (文档学习)
- **Multi-format Support**: PDF, DOCX, TXT document support
- **Knowledge Organization**: Generate knowledge directories, mark key points, record to classroom notes
- **Difficulty Annotation**: Annotate potential difficult points with detailed explanations in classroom notes

## Project Status

**Greenfield project** - This repository is newly initialized with only a requirements document (`需求文档.md`). The technology stack and architecture have not yet been established.
